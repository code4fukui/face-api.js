# face-api.js

ブラウザ上で動作する JavaScript の顔認識 API です。

## デモ
[デモページ](https://justadudewhohacks.github.io/face-api.js/)でfaceapi.jsの機能を体験できます。

## 機能
- 顔検出
- 顔ランドマーク検出
- 表情認識
- 年齢推定と性別推定

## 必要環境
- [face-api.js](https://github.com/justadudewhohacks/face-api.js) ライブラリ
- TensorFlow.js (ライブラリに含まれている)

## 使い方
### ブラウザ
最新のスクリプトをincludeするか、npmよりインストールできます。

### Node.js
必要なライブラリをインストールし、いくつかのpolyfillを設定する必要があります。
